# SG SAFECOIN – Code of Conduct

SG SAFECOIN is a global, humanitarian-focused digital currency project.
Our mission is to create a safer digital world for everyone.

To support this mission, all contributors and community members must follow this Code of Conduct.

---

# 1. Our Standards

### ✔ Respect  
Treat all contributors with kindness, respect, and professionalism.

### ✔ Inclusion  
All people are welcome, regardless of nationality, identity, or background.

### ✔ Collaboration  
We encourage constructive discussion, shared ideas, and teamwork.

### ✔ Integrity  
Ensure all contributions are honest, ethical, and aligned with the project's mission.

### ✔ Safety  
Do not engage in harassment, threats, or any form of harmful behavior.

---

# 2. Unacceptable Behavior

The following actions are prohibited:

### ❌ Harassment or discrimination  
### ❌ Promotion of scams, fraud, or malicious software  
### ❌ Attempts to obtain private SG SAFECOIN algorithms  
### ❌ Sharing misleading or harmful information  
### ❌ Personal attacks or abusive language  
### ❌ Unauthorized use of SG SAFECOIN branding  

---

# 3. Responsibilities of Maintainers

Project maintainers will:

- Enforce the code of conduct fairly  
- Remove harmful content or behavior  
- Protect community safety  
- Maintain high-quality public documentation  
- Safeguard the project's integrity  

Violations may result in warning, temporary bans, or permanent removal.

---

# 4. Scope

This code applies to:

- GitHub discussions  
- Issues and pull requests  
- Community posts  
- Project-related communication on any platform  

---

# 5. Enforcement

Report any violations by contacting project maintainers directly.

All reports will be:

- Reviewed quickly  
- Kept confidential  
- Acted upon fairly  

---

# 6. SG Philosophy

This project follows the Safeway Guardian principle:

### ⭐ "Serve, Guide, Save, Protect"  
Every contributor must behave in a way that reflects this mission.

---

# 7. Final Note

SG SAFECOIN is built to help humanity.  
Every contributor is part of that mission.  
Thank you for helping us build a safer digital future.
